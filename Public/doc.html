This API Will Allow Clients To Find Their Favorite Movies

This request will retrieve a list of movies or movie according to their genre.
URL HTTP method Parameters Request Response High-level description
/movies GET None None A JSON object holding data about a list of movies object This request will retrieve information about ten movies.
/movies/:genre GET genre: Genre of the movie None A JSON object holding data about a single movie object.
/movies/directors/:directorName GET directorName: Name of the director None A JSON object holding data about a single director This request will retrieve movies directed by specified director
/User POST None A JSON object holding data about a new user attempting to register A JSON object about a newly created user This request will allow clients to registar their name, id and favoriteMovie
/User GET None None A JSON object with a list of users The request retrieves user information by username
/User/:Username GET Username: Username of the user None A JSON object with the specified username This request retrieves user information by username
/Users/:Username: PUT Username: Username of the user A JSON object holding updated keys and values about single user. A JSON object holding updated user information. Updated user account information
/User/:Username/movies/MovieID POST Username: Username of the user. MovieID: ID of the movie to be added None A JSON object with updated favoritemovie. Adds a movie to the users list of favorite movies.
/User/:Username/movies/:MovieID DELETE Username: Username of the user. MovieID: ID of the movie to be deleted None A JSON object with the removed movie. Removes a movie from the users list of favoritemovies.
