This API Will Allow Clients To Find Their Favorite Movies

This request will return information about a given movie that is listed. A client will place the title endpoint in the URl and the server will provide information about that given movies such as year developed, director etc..
URL HTTP method Request Response High-level description
/movies GET None A JSON object holding data about all movies. This request will return a json object that holds information about ten movies.
/movies/:title GET None A JSON object holding data about a single movie, containing title, genre,description and year movie was filmed.
/movie/genre/:genreName GET None A JSON object holding data about a single genre based on type of genre This request will return information about a given genre that is listed. A client will place the genreName endpoint in the URl and the server will provide information about that given genre such as the type of genre such as actions, drama, comedy.
/movies/director/:directorName GET None A JSON object holding data about a single director, containing director name, bio, birth year, death year. This request will return information about a given director that is listed. A client will place the directorName endpoint in the URl and the server will provide information about that given director such birth date, movies directed and death date if applicable.
/users POST A JSON object holding data about a new user attempting to register A JSON object that allows new users to register This request will allow clients to registar their name, id and favoriteMovie
/users/:id PUT A JSON object holding data about the user which needs to be updated A JSON object that allows users to update their user information This URL endpoint will allow a client to update inforamtion about a user such as name, id and favoriteMovies
/users/:id/:movieTitle POST A JSON object holding data about single movie that users can add to their favorite list. A JSON object that allows users to add movie to their list of favorite movies. This URL endpoint will allow a client to add additional movies to their favoriteMovie lis.t
/user/:id/:movieTitle DELETE None A JSON object that allows users to delete a movie from their favorite movies list with a text response confirming student deleted movie. This URL endpoint will allow a client to delete a movie from their favoriteMovie list.
